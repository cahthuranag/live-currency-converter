Use it in your project

npm install live-currency-rates
import { Convert } from 'live-currency-rates';

const r = await Convert(100).from('USD').to('EUR');
console.log(r.amount); // 84.76

Three providers: Frankfurter (default, free), fawazahmed0 (free, 200+ currencies incl. crypto), AllRatesToday (real-time with API key). See the npm page for full docs.
